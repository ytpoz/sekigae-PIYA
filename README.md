# 席替えメーカー

ブラウザで動く、シンプルで盛り上がる席替えツールです。
特別なソフトのインストールは不要で、URLにアクセスするだけで誰でもすぐに席替えが始められます。

## 特徴

- **ドラムロール演出**: スタートボタンを押すとドラムロールが鳴り、ワクワク感を演出します。
- **前席希望への対応**: 「目が悪い」「静かに集中したい」など、前の方の席を希望する人を優先的に配置できます。
- **自動番号補完**: 参加者名を入力するのが面倒なときは、空欄のままスタートすれば自動的に番号（1, 2, 3...）で席替えを行います。
- **全画面表示**: プロジェクターなどで大きく映して、全員で結果を確認できます。
- **表出力機能**: 決まった結果をExcelなどに貼り付けやすい形式で出力できます。

## 使い方

1. 参加者の名前（または番号）を改行区切りで入力します。
2. 前方の席（オレンジ色の枠）に座ってほしい人がいれば、その名前を入力します。
3. 座席のレイアウト（横×縦）を調整します。
4. 席のマスをクリックすると、その席を「欠番（使用しない席）」に設定できます。
5. 「スタート！」ボタンで席替えを開始します。

## ファイル構成

- `index.html`: アプリ本体
- `ドラムロール.mp3`: 演出用音声
- `ロールの閉め.mp3`: 決定時音声

## 動作環境

最新のGoogle Chrome、Microsoft Edge、Safariなどのブラウザで動作します。

## 開発の背景

学校やイベント、社内のチームなどで、公平かつ少しだけ楽しく席替えができるツールが欲しくて作成しました。
